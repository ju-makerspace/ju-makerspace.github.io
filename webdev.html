course - Web Development

Description

Web development is the coding or programming that enables website functionality, per the owner's requirements. It mainly deals with the non-design aspect of building websites, which includes coding and writing markup.Web design refers to the design of websites that are displayed on the internet. It usually refers to the user experience aspects of website development rather than software development.

Who can learn this course?

* SOFTWARE ENGINEERING STUDENTS
* DESIGNING STUDENTS
* CREATORS
* MAKERS * CODERS

What will you learn?

* HOW TO MAKE WEB PAGES
* HOW TO CONNECT THROUGH LINKS AND DIFFERENT WEB PAGES
* RESPONSIVE DESIGNING
* HTML5,CSS3 AND BOOTSTRAP FRAMEWORK
* PROJECTS(TIC TAC TOE GAME DESIGN & SIMPLE WEBSITE)
